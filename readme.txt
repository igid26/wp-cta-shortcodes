"CTA Shortcodes in Post" is a free plugin that allows you to embed "Call to
Action" in articles and pages using a simple "Shortcode". Your "Call to
actions" feature can be placed directly in the text which increases
interaction. "Shortcodes" include 3 attributes: "title, text button and url
button"
The CTA feature can be graphically customized using the settings found in
the plugins administration. You can set the background color of the box,
button and text, also the alignment of the text and corner rounding.

Example usage: [cta_shortcodes title="" text-button="" url-button=""]

Advantages of
CTA directly in articles and pages
Increased CTR
Ability to customize the design
Ease of use



== Installation ==
1. Upload `wp-cta-shortcodes` folder to the `/wp-content/plugins/` directory or install the plugin from the WordPress plugin repository.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Go to the administration area Settings > CTA Shortcodes Settings.
1. Configure shortcodess.
